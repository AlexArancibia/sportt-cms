# sportt-cms

Panel CMS (Next.js) para administrar tiendas, productos, órdenes, cupones, kardex y estadísticas.

## Quickstart

1) Instalar dependencias:

```bash
pnpm install
```

2) Configurar variables de entorno (mínimo):

- `NEXT_PUBLIC_BACKEND_ENDPOINT`

3) Levantar el proyecto:

```bash
pnpm run dev
```
